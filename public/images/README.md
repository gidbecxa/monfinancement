# Images Directory

This directory contains all images used across the Mon financement platform.

## Directory Structure

### `/team`
Team member photos for the "Our Team" section on the home page.

**Required images:**
- `advisor.jpg` - Photo of advisory team members
- `controller.jpg` - Photo of management controller team members  
- `analyst.jpg` - Photo of expert analyst team members

**Recommended specs:**
- Format: JPG or PNG
- Size: 400x400px minimum (square aspect ratio)
- Professional headshots or team photos
- Optimized for web (< 200KB per image)

### `/partners`
Partner organization logos for the "Partners" carousel section.

**Required images:**
- `sg.png` - Société Générale logo (transparent background)
- `bnp.png` - BNP Paribas logo (transparent background)
- `hsbc.png` - HSBC logo (transparent background)
- `revolut.png` - Revolut logo (transparent background)
- `imf.png` - IMF logo (transparent background)

**Recommended specs:**
- Format: PNG with transparent background
- Size: 200x100px (2:1 aspect ratio)
- High resolution logos
- Optimized for web (< 100KB per image)

### `/testimonials`
Photos of beneficiaries who have provided testimonials.

**Required images:**
- `marie.jpg` - Photo for Marie D. (France)
- `ahmed.jpg` - Photo for Ahmed K. (Maroc)
- `sofia.jpg` - Photo for Sofia R. (Espagne)

**Recommended specs:**
- Format: JPG or PNG
- Size: 300x300px minimum (square aspect ratio)
- Natural, friendly photos
- Optimized for web (< 150KB per image)

### `/families`
Photos of happy families for the carousel section showcasing successful beneficiaries.

**Recommended images:**
- Multiple photos (5-10) of diverse families
- Showing happiness, gratitude, and positive outcomes
- Various settings (home, school, health facilities, etc.)

**Recommended specs:**
- Format: JPG
- Size: 1200x800px minimum (3:2 aspect ratio)
- High quality, emotionally engaging photos
- Optimized for web (< 300KB per image)

## Image Optimization

All images should be optimized before uploading:
- Use tools like TinyPNG, ImageOptim, or Squoosh
- Maintain good quality while minimizing file size
- Consider WebP format for better compression (with JPG/PNG fallbacks)
- Use responsive images with `srcset` for different screen sizes

## Copyright & Usage Rights

Ensure all images used have proper:
- Usage rights and licensing
- Model releases (for photos of people)
- Permission from organizations (for logos)
- Attribution where required

## Placeholder Images

Currently, the application uses icon-based placeholders. Replace these with actual images following the specifications above to complete the home page redesign.
